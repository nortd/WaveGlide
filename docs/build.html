WaveGlide Build Instructions

  1. order parts from BOM
  2. solder electronics -> video soldering guide
  3. flash firmware -> see general Trinket guide
  4. assemble tubing
  5. slide into case
  6. charge, go flying
  7. Tell us you've built one.
